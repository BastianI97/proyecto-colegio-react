REPOSITORIOS DEL PROYECTO
Proyecto: Libro de Clases Digital - Colegio Bernardo O'Higgins

Repositorio principal:
- https://github.com/Bastian197/proyecto-colegio-react

Frontend:
- Repositorio: https://github.com/Bastian197/proyecto-colegio-react
- Carpeta: Frontend-colegio
- Descripción: Componente frontend desarrollado con React/Vite. Contiene vistas para profesor, alumno y apoderado, incluyendo dashboard, horarios, calificaciones y asistencia.

Backend For Frontend:
- Repositorio: https://github.com/Bastian197/proyecto-colegio-react
- Carpeta: Backend-colegio/bff-colegio
- Descripción: BFF encargado de centralizar la comunicación entre el frontend y los microservicios de notas y asistencia.

Microservicio de Notas:
- Repositorio: https://github.com/Bastian197/proyecto-colegio-react
- Carpeta: Backend-colegio/ms-notas
- Descripción: Microservicio encargado de registrar, consultar, actualizar, eliminar notas y calcular promedios por alumno.

Microservicio de Asistencia:
- Repositorio: https://github.com/Bastian197/proyecto-colegio-react
- Carpeta: Backend-colegio/ms-asistencia
- Descripción: Microservicio encargado de registrar, consultar, actualizar, eliminar asistencias y calcular resúmenes de asistencia por alumno.

Backend base:
- Repositorio: https://github.com/Bastian197/proyecto-colegio-react
- Carpeta: Backend-colegio/auth-servise
- Descripción: Servicio backend base existente del proyecto, con estructura Spring Boot, Maven, JPA y MySQL.

Arquetipo/Base Maven:
- Repositorio: https://github.com/Bastian197/proyecto-colegio-react
- Carpeta: Backend-colegio/arquetipo-base-maven
- Descripción: Documentación de la base Maven utilizada para construir los componentes backend del proyecto.
